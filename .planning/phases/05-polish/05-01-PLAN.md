# Phase 5 Plan 01: Animations & Visual Feedback

## Objective
Add satisfying animations and visual feedback throughout the app to make it feel polished and fun.

## Context
@Buzzword Bingo/Buzzword Bingo/Views/BingoSquareView.swift
@Buzzword Bingo/Buzzword Bingo/Views/BingoGridView.swift
@Buzzword Bingo/Buzzword Bingo/Views/HandoffView.swift

## Tasks

### Task 1: Enhanced Square Tap Animation
**Type:** modify
**Files:** `Buzzword Bingo/Buzzword Bingo/Views/BingoSquareView.swift`
**Action:**
- Add bounce animation on mark (spring effect)
- Add checkmark or stamp visual that appears briefly
- Add subtle haptic feedback (on iOS)
- Make the green fill animate in with a satisfying sweep

**Verify:** Tapping a square feels satisfying

### Task 2: Winning Line Celebration
**Type:** modify
**Files:** `Buzzword Bingo/Buzzword Bingo/Views/BingoSquareView.swift`, `BingoGridView.swift`
**Action:**
- Winning squares pulse/glow animation
- Staggered animation along the winning line
- Maybe confetti particles near winning squares

**Verify:** Getting bingo feels exciting

### Task 3: Handoff Screen Polish
**Type:** modify
**Files:** `Buzzword Bingo/Buzzword Bingo/Views/HandoffView.swift`
**Action:**
- Player name slides/fades in with drama
- "Ready" button has a subtle pulse
- Background has subtle animated gradient or particles

**Verify:** Handoff feels like a moment

### Task 4: Build Verification
**Type:** verify
**Action:** Build both platforms
**Verify:** Both platforms compile, animations work

## Verification
- [ ] Square tap feels satisfying
